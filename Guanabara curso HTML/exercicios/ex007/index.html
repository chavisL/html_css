Código-fonte

Para exibir código-fonte em HTML, você pode usar a tag <code> para trechos curtos de código e a tag <pre> para blocos de código preservando espaços e quebras de linha, como vistos em códigos como PHP, JS, etc.

Code utiliza estrutura monoespaçada (mesma distância entre as letras para todas as letras )

Ex (code): document.getElementById
Ex (pre): 
    if (gabriel == homem)
    {
    echo "Gabriel não é pra ser homem, mentira isso ai";
    } else {
    echo "Gabriel não é um homem... fato!";
    }

Pode-se mesclar também o pre com o code!

    
    if (gabriel == homem)
    {
    echo "Gabriel não é pra ser homem, mentira isso ai";
    } else {
    echo "Gabriel não é um homem... fato!";
    }

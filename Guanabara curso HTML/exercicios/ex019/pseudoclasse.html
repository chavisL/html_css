Pseudo Classe

É uma palavra-chave que é adicionada após uma tag e dois pontos como por exemplo: div:hover

(Div é a tag. Os dois pontos com a palavra-chave é o nosso estado ESPECIAL que criamos para aquela classe... Então Classe + Estado especial = PseudoClasse)

Explicando

id

Para id só se pode usar em um único elemento, funciona se colocar em outros, mas a W3C recomenda só um para um elemento.

Procure sempre colocar a tag e depois o nome da sua id assim: h1#principal

Hieraquicamente id vem primeiro que class

class

Para o class, você pode aplicá-lo em mais de um elemento, desde que possar compartilhar as mesmas propriedades, como color, background-color, etc.

Você pode usar mais de um class para um único elemento é só escrever dando espaço: class="destaque explicando" (destaque e explicando são classes diferentes)

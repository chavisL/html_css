Semântica e forma

Na web, a semântica refere-se ao significado e à estrutura do conteúdo, enquanto a forma diz respeito à aparência visual desse conteúdo. Utilizar elementos HTML semânticos, como <header>, <nav>, <article> e <footer>, ajuda a definir claramente o propósito de cada seção da página, facilitando a compreensão tanto para os usuários quanto para os mecanismos de busca. HTML deixou de ter tags para formas e agora no HTML5 foca na semântica!

Em resumo Semântica é focado no significado. Já a Forma é focada na aparência visual.

Porém se estiver em dúvida pensando assim: "Então não pode deixar o site bonito então?", acalme-se, pois esse é o trabalho do CSS.

Principais formatações

Negrito e Destaque

Para aplicar o NEGRITO basta utilizar a tag <b> (Não semântico, foco no visual)

Para apliciar o DESTAQUE basta utilizar a tag <strong> (Semântico, foca em destacar)

Itálico e Ênfase

Para aplicar o ITÁLICO basta utilizar a tag <i> (Não semântico, foco no visual)

Para aplicar a ÊNFASE basta utilizar a tag <em> (Semântico, foca em enfatizar)

Texto marcado

Para aplicar a MARCAÇÃO basta utilizar a tag <mark>

Texto grande e pequeno

Para aplicar o texto GRANDE basta utilizar a tag <big> (Esta obsoleta, ou seja, não vai ser mais utilizada nas próximas atualizações)

Para aplicar o texto PEQUENO basta utilizar a tag <small> (Este não esta obsoleto, pois existem letras miudas em contratos, e ainda é considerado)

Texto deletado

Para aplicar o texto DELETADO basta utilizar a tag <del> (Deve ser lido, mas não considerado)

Texto inserido

Para aplicar o texto INSERIDO basta utilizar a tag <ins> (Semântico, deve ser considerado)

Texto com SUBLINHADO pode ser aplicado utilizando a tag <u> (Não semântico, foco no visual)

Subscrito e Sobrescrito

Para aplicar o texto em SUBSCRITO basta utilizar a tag <sub> (Ex: H2O)

Para aplicar o texto em SOBRESCRITO basta utilizar a tag <sup> (Ex: 103 = 1000 )
